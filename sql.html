Basic

  • Basic SELECT Queries
    • Example: SELECT column_name FROM table_name;

  • Filtering with WHERE
    • Example: SELECT * FROM table_name WHERE condition;
    • Use cases: Pulling specific rows based on conditions.
  • Sorting with ORDER BY
    • Example: SELECT * FROM table_name ORDER BY column_name ASC/DESC;
    • Use cases: Sorting data in ascending or descending order.
  • Using DISTINCT
    • Example: SELECT DISTINCT column_name FROM table_name;
    • Use cases: Finding unique values in a column.
  • Basic Aggregations
    • Examples: COUNT, SUM, AVG, MAX, MIN.
    • Use cases: Calculating totals, averages, or other metrics.
  • LIMIT / TOP (or FETCH FIRST)
    • Examples: SELECT * FROM table_name LIMIT 10; or SELECT TOP 10 * FROM table_name;
    • Use cases: Retrieving a specific number of rows.
  • Basic Joins
    • Example: SELECT * FROM table1 INNER JOIN table2 ON table1.id = table2.id;
    • Use cases: Combining data from two related tables.

    • Example: INSERT INTO table_name (column1, column2) VALUES (value1, value2);
    • Use cases: Adding new records to a table.
  • Updating Data
    • Example: UPDATE table_name SET column1 = value1 WHERE condition;
    • Use cases: Modifying existing records.
  • Deleting Data
    • Example: DELETE FROM table_name WHERE condition;
    • Use cases: Removing specific rows from a table.
  • Basic Functions
    • Examples: UPPER, LOWER, LEN, ROUND.

  • Using Aliases
    • Example: SELECT column_name AS alias_name FROM table_name;
    • Use cases: Making column or table names easier to read.

    • Example: SELECT * FROM table_name WHERE column_name LIKE 'pattern%';

    • Use cases: Understanding the schema of a table.
  • Simple Case Statements
    • Example: SELECT column1, CASE WHEN condition THEN result END AS new_column FROM table_name;
