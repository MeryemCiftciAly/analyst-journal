HR Analytics: Workforce Trends and Diversity Insights

This project examines workforce trends, employee retention, and performance metrics while also exploring race and gender distribution across departments. By using Excel for data cleaning and Tableau for visualization, I identified key patterns in attrition, job satisfaction, and disparities that could inform equitable hiring and HR policies. The insights from this analysis can help organizations address discrimination risks, improve diversity efforts, and support data-driven decision-making.
