Data Visualizations

Scatterplot: Sepal Width vs. Sepal Length

A scatterplot was used to visualize the relationship between two variables in the dataset to identify patterns, trends, or correlations. Each data point represents an individual observation, helping to capture the overall distribution and relationship between variables.

 ```{r echo=TRUE, message=FALSE, warning=FALSE, error=FALSE}
library(ggplot2)

#Custom color pallette

custom_colors <- c("setosa" = "purple", "versicolor" = "orange", "virginica" = "green")
my_plot <- ggplot(iris, aes(x = Sepal.Length, y = Sepal.Width, color = Species))+
  geom_point(size = 4) +
  labs(title = "Sepal Length vs Sepal Width by Species",
       x= "Sepal Length (cm)",
       y= "Sepal Width (cm)") +
  scale_color_manual(values = custom_colors) +
theme(panel.grid.major = element_blank(), panel.grid.minor = element_blank(), 
      panel.background = element_blank()) +
        theme_classic()

## View the plot to show in the output
print(my_plot)

ggsave("Sepal_LvWScatterplot.png", plot = my_plot, width = 4, height = 6)
```

Boxplot: Petal Length

Box Plot: A box plot was used to summarize the distribution of a single variable, showing key statistical measures such as the median, quartiles, and range. It highlights the concentration of data within the values that fall in the middle 50% of the data, not too high and not too low—just the main chunk of the data. It also identifies outliers which are data points that stand out because they are much higher or lower than the rest of the data set and noticeably stand out.
